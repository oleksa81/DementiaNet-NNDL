# DementiaNet-NNDL
CS 7357 Neural Networks and Deep Learning Research Group Project:
Speech-based Dementia Assessment

Using log-mel spectograms for dementia detection

 ## Project overview
- Using a speech processing model wav2vec to extract audio features from
spontaneous speech clips to assess the levels of dementia
### Motivation
- Traditional diagnostic methods rely heavily on clinical tests
- clinical interviews, neuropsychological tests, or imaging techniques
- resource-intensive and inconvenient
- Speech-based assessment => Scalability; accessibility; personalization
### Rationale
- Dementia is a progressive neurodegenerative condition characterized by cognitive
decline, which affects memory, language, and other cognitive abilities
- Speech impairments, including changes in fluency, prosody, vocabulary, and
sentence structure, are among the earliest indicators of cognitive decline
- Semantic deficits (e.g., difficulty finding words)
- Syntactic impairments (e.g., challenges in forming coherent sentences)
- Pragmatic changes (e.g., difficulty maintaining conversational flow)
- Acoustic alterations (e.g., changes in speech rate, articulation, patterns of stress and
intonation)
